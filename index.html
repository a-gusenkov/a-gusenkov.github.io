Neurodiversity and Software Development

Semester-long solo honors project for Human-Computer Interaction course. Researched the effect of Dyslexia and ADHD on the usability of computer software to develop a prototype game design that teaches software developers about accessible programming design and practices. Conducted qualitative and quantitative research by running student interviews, creating and distributing surveys, and talking to subject matter experts. Created the game prototype with the use of Figma. Conducted user testing to receive feedback and alter elements of the original game prototype.
